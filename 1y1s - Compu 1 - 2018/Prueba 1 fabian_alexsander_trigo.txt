1) Describir conceptos
  Linux: Sistema Operativo de codigo abierto en el cual varias personas
  aportan a mejorarlo
  usuario: Es aquella entidad que utiliza el servicio electronico,
  aquel que le da ordenes al Terminal
  cat: Es un comando para leer archivos de texto dentro del Terminal de Linux
  vi: Es un editor de texto un tanto mas avanzado dentro del terminal,
  el cual permite editar y crear archivos de texto.
  rm -r: O su otra funcionalidad rmdir, rm viene de remove, es el comando para
  borrar archivos, y con la opcion -r sirve para borrar carpetas contenedoras
  de archivos.
  head: Similar al comando de lectura de texto cat, sin embargo sirve para
  ver las primeras lineas de texto de un archivo omitiendo las otras.
  ls -a: ls proviniente de list, y la opcion -a de all, sirve para ver todos
  los archivos incluyendo los ocultos dentro de un directorio.
